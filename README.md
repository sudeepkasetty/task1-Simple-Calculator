# Simple Calculator (Java)

## Description

This is a console-based calculator application developed in Java. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The program uses methods for each operation, accepts user input through the `Scanner` class, and allows multiple calculations using a `do-while` loop until the user chooses to exit.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* Repeats calculations until the user exits

## Technologies Used

* Java
* Scanner Class
* Methods
* Switch Case
* Do-While Loop

## How to Run

1. Compile the program:

   ```
   javac simpleCalculator.java
   ```
2. Run the program:

   ```
   java simpleCalculator
   ```
3. Select an operation, enter two numbers, and choose whether to continue or exit.

## Output

The program displays the result of the selected arithmetic operation and prompts the user to perform another calculation until they choose to exit.
